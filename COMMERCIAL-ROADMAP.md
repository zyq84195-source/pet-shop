## 技术架构需求

### 后端服务
- Node.js (Express/Nest.js) 或 Java (Spring Boot)
- 数据库：MySQL + Redis
- 消息队列：RabbitMQ（订单/通知）
- 定时任务：订单超时取消、数据统计

### 第三方服务
- 阿里云 OSS（图片存储）
- 阿里云短信（验证码/通知）
- 支付宝开放平台
- 微信开放平台
- 物流查询 API

### 安全合规
- ICP 备案
- 等保认证（如需要）
- 隐私政策
- 用户协议
